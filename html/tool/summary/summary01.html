개요

  1. Home
  2. 개발 도구
  3. 개요

본 가이드는 S/W ALM(Application Lifrcycle Management) 관점에서 활용 할 수 있는 개발 지원 도구별 소개, 설치, 주요기능 및 활용 방법에 대한 가이드 제공을 통해 프로젝트 진행의 용이를 제공함을 목적으로 작성되었습니다.

Toolchain 구성

개발도구는 목적에 따라 총 10개의 항목으로 Toolchain을 형성하고, 각 항목을 기준으로 도구를 선정 및 사용 가이드를 작성하였습니다.

구성 항목 주요 기능 선정 도구
프로젝트 관리 일정, 진도, 작업 및 이슈관리 Redmine
요구사항 관리 도구 요구사항 변경관리, 추적관리 OSRMT
설계 도구 UML 지원 StarUML
형상관리 SW 소스 및 자원 관리 Git
빌드 자동빌드 Maven
저장소 Nexus
소스 진단 도구 정적분석 - 코딩표준, 보안 취약점 점검 SonarQube
단위 테스트 도구 테스트 코드 작성, 관리, 단위 기능 검증 Junit
통합 테스트 도구 테스트 실행 및 결과 관리 및 보고 Fitness
부하 테스트 도구 부하 테스트 Jmeter
지속 통합 도구 빌드 스케줄 및 배포 관리 Jenkins

구성

개발 도구별 구성 형식은 아래와 같이 구성됩니다.

구성 형식 설명
개요 도구 소개, 라이선스 타입, 특징, 주요 기능 및 설치 환경에 대한 정보를 제공합니다.
설치 사전 설치 도구 및 단계별 설치 단계 정보를 제공합니다.
주요 기능 도구의 주요 기능을 소개합니다.
활용 예제 예제를 바탕으로 한 활용 방법을 소개합니다.
